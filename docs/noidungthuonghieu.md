## Cách dán vào Payload

Mỗi hãng dưới đây gồm 2 phần:
- **Tóm tắt** → dán vào field `summary` (dùng cho card + meta description, nên giữ ngắn).
- **Bài viết** (markdown) → dán vào field `content` (rich text).

**Mẹo dán rich text (Lexical của Payload):**
- Editor Lexical thường tự nhận markdown: gõ/dán `## ` → Heading 2, `### ` → Heading 3, `**đậm**` → in đậm, `- ` → bullet, `> ` → trích dẫn.
- Nếu dán cả đoạn mà không tự convert: dán dạng văn bản thuần rồi bôi chọn từng dòng và chọn Heading 2 / Heading 3 / Bullet list / Quote tương ứng theo cấu trúc bên dưới.
- `##` = Heading 2, `###` = Heading 3. Không dùng Heading 1 trong content (H1 đã là tiêu đề trang).
